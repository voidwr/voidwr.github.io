OxFun CTF 2026 - Lookup

Back to 0xFun Back to Writeups

Overview

Lookup was a fairly easy OSINT challenge that required leveraging DNS records.

Challenge

The challenge prompt said that the event takes place on the host website and that we needed to search for it.

Prompt
        
        This event takes place on ctf.0xfun.org, but you can easily find it by searching.

Flag

Trying various commands nslookup, ping etc. and even inspecting the source code of the website led to no results. However, by using dig to query for TXT DNS records revealed a TXT record that contained the flag.

Flag: 0xfun{4ny_1nfo_th4ts_pub1cly_4cc3ss1bl3_1s_0S1NT}
