OxFun CTF 2026 - Perceptions

Back to 0xFun Back to Writeups

Overview

Perceptions was an easy web challenge that costed 150 pts at the time of completion. It didn't require any complex exploitation techniques, just a bit of careful reading.

Challenge

We had to take a look at a blog and find some hidden attributes.

Prompt
        
        Take a look at the blog I created! It has a neat backend and, interestingly, 
          seems to use fewer ports.

Flag

When we start the challenge and go to the given link we land on a blog page with some posts. Notably, the owner of the blog (Charlie) has named this blog "Charlie's Blog".

Figure 1: Initial view of the blog page.

By inspecting the Pages tab on the left we can see that there are many clues about what we have to do. Let's start with the My Server page.

Figure 2: My Server page.

The My Server page contains a hint about the ports used by the machine. Specifically, it says that it is a full Linux container but it is only used for web serving and some pretty generic Linux remote access stuff. From that we can infer that the machine has 2 primary ports open:

  • Port 80 for HTTP
  • Port 51230 for SSH since that's what the server provided us with from the prompt

Let's make a mental note of this and take a look at the Secret Post for now.

Figure 3: Secret Post page.

The Secret Post contains a hint. It says that a password is hidden in this page. From this we can understand that something is hidden on the source code of the page, since you can write comments on HTML documents that are not visible on the version the users see. So Let's inspect the source code of the page.

Figure 4: HTML comment found secret.

We can see UlLOPNeEak9rFfmL and we can use it to login. But the login page is not ready so we we can use it elsewhere like in the SSH port. Let's try to login with the username charlie and the password we found.

Figure 5: SSH login.

We can see that we have access to the machine. Let's check the home directory of charlie with and find the flag in the /secret_flag_333 directory.

Figure 6: Flag: 0xfun{p3rsp3c71v3.15.k3y}
